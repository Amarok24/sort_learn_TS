# sort_learn_TS
TypeScript learning project. BubbleSort using different types of data. Code version 1.

Run with DENO
```
$ deno run build/index.js
hello from index.ts
[ -5, 0, 3, 10 ]
aabXy
-10
-3
4
500
```

Run with NodeJS
```
$ node build/index.js
hello from index.ts
[ -5, 0, 3, 10 ]
aabXy
-10
-3
4
500
```
